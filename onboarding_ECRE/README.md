# 이크레더블 신규입사자 가이드북

## 📖 프로젝트 소개

이크레더블 신규입사자를 위한 온보딩 가이드북 웹사이트입니다.

## 🚀 빠른 시작

1. `index.html` 파일을 웹 브라우저로 열기
2. 또는 웹 서버에 업로드하여 사용

## 📁 파일 구조

```
cursor/
├── index.html              # 메인 웹페이지 파일
├── styles.css              # 스타일 파일 (디자인, 레이아웃)
├── script.js              # 동작 관련 파일 (메뉴, 애니메이션 등)
├── fonts/                 # 폰트 파일들 (Paperlogy)
├── ecredible-b-kr-crop.png # 로고 이미지
├── README.md              # 이 파일
└── MAINTENANCE_GUIDE.md   # 유지보수 가이드 (상세 설명)
```

## 🛠️ 유지보수

**비개발자 관리자를 위한 상세 가이드는 `MAINTENANCE_GUIDE.md` 파일을 참고하세요.**

### 주요 작업

- **텍스트 수정**: `index.html` 파일을 열고 원하는 텍스트를 찾아 수정
- **컨텐츠 추가**: 각 섹션에 있는 템플릿 주석을 참고하여 추가
- **이미지 추가**: `images/` 폴더에 이미지를 저장하고 HTML에 경로 추가

## 📝 주요 섹션

1. **회사소개** - 회사 개요, 비전, 법인관계도
2. **환경설정** - 인수물품, 이중망, 설치 S/W, 출입등록
3. **회사제도 안내** - 직급, 인사평가, 복리후생, 노동조합
4. **시설 안내** - 본사 및 지역사무소
5. **기타 안내사항** - 계정, 명함, ECM, 원격근무 등
6. **FAQ** - 자주 묻는 질문

## 🎨 디자인

- **폰트**: Paperlogy (9가지 두께)
- **색상**: Primary Color (#000049)
- **반응형**: 모바일, 태블릿, 데스크톱 지원

- 변경 후 브라우저에서 테스트하세요
